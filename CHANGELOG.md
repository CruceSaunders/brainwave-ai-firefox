# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-03

### Added
- Initial release
- **Summarize** - Condense any text into key points
- **Rewrite** - Improve clarity and professionalism
- **Explain** - Simplify complex topics (ELI5, Simple, Detailed modes)
- **Translate** - Convert between 10+ languages
- Context menu integration
- Popup interface with copy to clipboard
- Privacy-first design (no tracking, no data storage)
- Powered by Claude AI (Anthropic)
- Firefox Manifest V2 compatibility
